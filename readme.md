# 🧠 AutismoXR – Meu Primeiro Ambiente VR

## 📌 Descrição do Projeto

Este projeto consiste na criação de um ambiente em Realidade Virtual (VR) desenvolvido no Unity, com foco em simular um espaço interativo voltado à sensibilização e compreensão de experiências relacionadas ao autismo.

O ambiente foi construído com base nos fundamentos de XR (Extended Reality), permitindo navegação e interação dentro de um cenário virtual estruturado.

---

## 🎯 Objetivo

Demonstrar, na prática, a aplicação dos conceitos fundamentais de desenvolvimento em XR, incluindo:

- Configuração de ambiente VR no Unity
- Uso do Meta XR SDK
- Criação de cenário navegável
- Implementação de interação com objetos

Além disso, o projeto busca representar um contexto significativo dentro do metaverso, com potencial educacional e de conscientização.

---

## 🌍 Contexto no Metaverso

O ambiente foi pensado como um espaço imersivo que simula situações e estímulos sensoriais, possibilitando maior empatia e compreensão sobre diferentes percepções do ambiente.

Esse tipo de aplicação pode ser utilizado em:

- Treinamentos
- Educação
- Sensibilização social
- Experiências imersivas interativas

---

## 🛠️ Tecnologias Utilizadas

- Unity (Engine principal)
- Meta XR SDK
- XR Interaction Toolkit
- C# (para interações)

---

## 🧱 Estrutura do Projeto

O repositório está organizado da seguinte forma:

- `Assets/` → Scripts, modelos 3D, materiais e cenas
- `Packages/` → Dependências do Unity
- `ProjectSettings/` → Configurações do projeto

---

## 🎮 Funcionalidades

- Ambiente 3D navegável
- Objetos posicionados de forma coerente no cenário
- Interação com elementos do ambiente
- Estrutura pronta para expansão com novas interações

---

## ⚙️ Execução

Para executar o projeto:

1. Abra o Unity Hub
2. Adicione este projeto
3. Abra com a versão compatível do Unity
4. Execute pelo modo Play (Editor)
